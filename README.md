# InfluenceKit Claude Plugin

Connect Claude to your InfluenceKit account to manage campaigns, track performance, troubleshoot deliverables, and prepare reports.

## Prerequisites

- [Claude Code](https://claude.ai/claude-code) v1.0.33+
- An InfluenceKit admin account
- InfluenceKit MCP connection configured in Claude Desktop

## Installation

**Step 1** — Add the InfluenceKit marketplace:

```
/plugin marketplace add InfluenceKit/claude-plugin
```

**Step 2** — Install the plugin:

```
/plugin install influencekit
```

## What you get

After installing, Claude knows how to use all 10 InfluenceKit MCP tools to help you:

- Check campaign performance and deliverable stats
- Diagnose why a post isn't showing stats
- Fix broken deliverables (clear errors, refresh stats)
- Review reports before sharing with clients
- Audit social connection health

## Example usage

Once installed, just ask Claude naturally:

- "How is our summer campaign doing?"
- "Why isn't this Instagram post showing any stats?"
- "Check if all our social connections are healthy"
- "Review my Q1 report for any issues before I send it"
